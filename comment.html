HTML Comment

An HTML comment begins with <!-- and the comment closes with -->. You can use a comment to further explain the purpose of the code, or to set a reminder on what to do next.

Code Example

The default display of HTML comment is inline.

<!-- I am the default display -->

But you can still make a multi-line comment like this

<!-- I am a
multi-line
comment
-->
