1.3.2

DOCTYPE

DOCTYPE stands for Document Type Declaration. It tells the web browser what version of HTML to render.

DOCTYPE declaration must be place at the very beginning of the web document. It must be written inside the angle bracket like this, <!DOCTYPE html >. Though it is inside an angle bracket like any HTML elements, DOCTYPE is not a tag rather it's a statement.

Take note that it's syntax is not strict or case sensitive like the previous version of HTML.

Code Example

HTML5 DOCTYPE Declaration

<!DOCTYPE html >

It is also possible to be written like this

<!doctype html>

This way is also possible

<!docType>

DOCTYPE in Other Versions of HTML

Versions prior to HTML5 were based on Standard Generalized Markup Language (SGML), so their !doctype declaration had to contain a reference to their corresponding document type declaration (DTD).

Code Example

HTML 4.01 DOCTYPE Declaration

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd" >

XHTML 1.0 Strict DOCTYPE Declaration

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd" >
