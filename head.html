1.3.5

HTML Head Tag

The <html> tag represents the root of an HTML document.

The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag).

Code Example

<!DOCTYPE html>

<html lang="en">

<head>

<title>TITLE</title>

</head>

<body>

<h1>This is a heading</h1>

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Consectetur, magnam blanditiis corporis inventore a ipsum nemo fugit asperiores fuga voluptates? </p>

</body>

</html>
